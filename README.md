📚 Book Donate Platform
The Book Donate Platform is a web-based system where users can donate books and earn coins as rewards. These coins can later be redeemed for exciting gifts. Admins manage the available gifts and set coin requirements for redemption. This platform promotes book sharing and rewards active contributors.

🚀 Key Features
🔐 User Registration & Authentication
Email verification during registration.

Secure login and logout functionality.

Only verified users can log in.

Mobile number is required and must be unique.

🎁 Gift Redeem System
Admin can add and manage gift items.

Admin sets how many coins are required to redeem each gift.

Users can view available gifts and redeem them using coins.

👤 User Profile Management
Users can view and update their profile.

Ability to change password.

Profile includes mobile number (unique for each user).

📚 Book Donation
Users can list books for donation with all necessary details.

Ability to update or manage donated books.

💬 Platform Reviews
Users can leave comments or feedback on the platform to support community engagement.

☁️ Deployment
The application is deployed on a secure and scalable hosting service.

Fully configured for live use with environment-specific settings.

🛠️ Technologies Used
Backend: Django (MVT Pattern)

Frontend: HTML, CSS, Bootstrap, Tailwind CSS

Database: MySQL

Hosting: Deployed on https://book-donate-platform.onrender.com/


▶️ How to Run Locally
Clone the repository:


git clone https://github.com/yourusername/book-donate-platform.git
cd book-donate-platform
Install dependencies:


pip install -r requirements.txt


Apply migrations:
python manage.py migrate
Run the development server:

python manage.py runserver
Access the site at http://localhost:8000
